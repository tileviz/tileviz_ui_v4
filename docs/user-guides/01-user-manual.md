# User Manual

## Overview

TileViz lets you visualize how tiles will look in your room in real-time 3D before purchasing. You can configure the room type, dimensions, tile sizes, and apply different tiles to different walls and floors.

---

## Screens

### 1. Visualizer
The main 3D view of your room.

**What you can do:**
- Rotate the camera — drag/swipe to orbit around the room
- Zoom — pinch or scroll wheel
- Change room type (bathroom, kitchen, bedroom, balcony, parking)
- Change room dimensions (width, length, height in feet)
- Pick a tile size
- Change wall color
- Save the design
- Share as PDF

**Controls (3D canvas buttons):**
| Button | Action |
|---|---|
| 🏠 Interior | Toggle front wall visibility |
| ↺ / ↻ | Rotate left / right |
| + / - | Zoom in / out |
| ↺ Reset | Reset camera to default position |
| 💡 Light | Toggle lighting mode |
| 📦 Objects | Toggle room fixtures visibility |

---

### 2. Catalog
Browse all available tiles and assign them to your room zones.

**What you can do:**
- Filter tiles by room type (All / Bathroom / Kitchen / Bedroom / Balcony / Parking)
- Filter by tile size chips
- Search by tile name
- Tap a tile to assign it to the currently selected zone
- Open Zone Arena (sidebar) to configure which wall/row gets which tile

---

### 3. Zone Arena (inside Catalog)
Assign different tiles to different parts of your room.

**Basic flow:**
1. Open sidebar → select room type → set dimensions
2. Tap a zone row (e.g. "Row 1" on Walls or "Floor Tile")
3. The catalog grid highlights — tap any tile to assign it
4. The 3D view updates immediately

See [Zone Arena Guide](05-zone-arena-guide.md) for full details.

---

### 4. Saved Designs
All your saved 3D visualizations.

**What you can do:**
- Search by name, tile, or room type
- Filter by room type
- Load a design back into the Visualizer
- Delete a design

---

### 5. Inventory
Saved design library shared within your shop.

**What you can do:**
- Browse all inventory items
- Filter by room type and status (Active / Draft / Archived)
- Load any inventory design into the Visualizer
- View full details of an item

---

## Saving a Design

1. Configure your room in the Visualizer or Catalog screen
2. Tap the 💾 Save button
3. Enter a name for the design
4. Tap Save — the design is saved with a 3D thumbnail

---

## Sharing as PDF

1. Complete your room design
2. Tap the Share / PDF button in the Visualizer
3. A PDF is generated with room details and tile information
4. Share via email, WhatsApp, or save to device

---

## Tutorial

On first login, an interactive tutorial guides you through:
1. Selecting a room type
2. Browsing the catalog
3. Filtering by bathroom
4. Selecting a tile
5. Saving a design

You can skip the tutorial at any time. It only runs once.
