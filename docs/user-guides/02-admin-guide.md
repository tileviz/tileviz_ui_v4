# Admin Guide

Admins have full platform access — they can manage all shops, users, and the tile catalog.

---

## Dashboard

The admin dashboard shows:
- **Total Users** — all registered users across all shops
- **Active Shops** — shops currently on a plan
- **Tiles in Catalog** — approved tiles available
- **Pending** — tile catalog requests awaiting review

---

## User Management

Located in the Dashboard under "User Management".

**What you can do:**
- View all users (name, email, role)
- Toggle a user Active / Inactive
  - Inactive users cannot log in
  - Admins cannot be toggled

**Roles shown:**
- 👑 Admin
- 🏪 Shop Owner
- 💼 Sales Person

---

## Shop Management

Located in the Dashboard under "Registered Shops".

**What you see per shop:**
- Shop name and owner name
- Number of sales persons
- Plan badge (Starter / Pro / Enterprise)

---

## Admin Screen

The Admin tab provides deeper management:
- View and manage all shops
- View catalog tiles across all shops
- Manage platform-level settings

---

## Plan Badges

| Plan | Style |
|---|---|
| Starter | Grey — basic access |
| Pro | Gold — advanced features |
| Enterprise | Blue — full platform access |

---

## Notes

- Admins see ALL inventory across ALL shops
- Admins can delete any inventory item
- Admins cannot be made inactive by other admins
