# Sales Person Guide

Sales persons use TileViz to create and show tile visualizations to customers.

---

## Daily Workflow

```
1. Open TileViz
2. Go to Catalog → Open Zone Arena
3. Select room type that matches customer's room
4. Enter room dimensions (ask customer)
5. Assign tiles to walls and floor
6. Show customer the 3D view in Visualizer
7. Adjust tiles based on feedback
8. Save design or share as PDF
```

---

## Creating a Visualization

1. Tap **Catalog** tab
2. Tap **Zone Arena** (sidebar icon)
3. Select room type (Bathroom / Kitchen / Bedroom / Balcony / Parking)
4. Set dimensions (width × length × height in feet)
5. Choose tile size
6. Tap zone rows to assign tiles
7. Browse catalog grid and tap a tile to apply it
8. View result in the **Visualizer** tab

---

## Requesting a Tile

If a tile a customer wants is not in the catalog:
1. Go to Catalog
2. Find the tile (it may show as "not in catalog")
3. Tap **Request** to send it to your shop owner for approval
4. Once approved, the tile appears in your catalog

---

## Saving Designs

**Save as Design** (personal):
- Tap 💾 → enter name → Save
- Appears in your Saved Designs
- Only you can see it

**Save to Inventory** (shared with shop):
- Tap Save → Save to Inventory → enter name
- Visible to your shop owner and team
- Useful for showing popular combinations to future customers

---

## Sharing with a Customer

1. Complete the visualization
2. Tap the Share / PDF button
3. A PDF is created with:
   - Room type and dimensions
   - Tile names and sizes
   - 3D screenshot
4. Share via WhatsApp, email, etc.

---

## Loading a Previous Design

1. Go to **Inventory** tab
2. Find the design
3. Tap **Load**
4. It opens in Visualizer — you can modify and re-save

---

## Notes

- Sales persons cannot delete inventory items (only view and load)
- Cannot see other shops' data
- Cannot approve/reject tile requests (that's the shop owner's job)
